question
  id - string used that associates the answer with this value. i.e. "How many milligrams?" with id: "mg" would have mg: 2.5. 
  text - string containing the question to ask the user. does not contain type indications (i.e. no "(Y/N)")
  type:
    time - military time. indicator "(00:00 - 23:59): "
    yes_no - answer must be either "y","yes", or "n", "no" ignoring case. indicator "(Y/N): "
    int indicator "(Whole Number. min: {min}, max: {max}): "
    float indicator "(Float. min: {min}, max: {max}): "
    string
    enum indicator changes per enum ex: "valid answers:\n 1) answer1\n 2) answer2\n 3) answer3\n (1 - 3): "
  min: for floats and ints. min value that can be inputed
  max: for floats and ints. max value that can be inputed
  sub_questions: - questions to ask based on the users answer to a yes_no question.
    yes: - when answered yes can be empty
      - question1 
      - question2
    no: - when answered no can be empty
      - question1
      - question2
  answers: required for all enums. list of strings that are valid answers.
  ask_for_count: for ints. list of questions to ask. will ask each questions n amount of times. n is the answer given by the user. the id for each one is the current iteration + 1.
