# OneiroDump

A C# cli tool that makes the process of documenting sleep and dream data easier.

# In Progress

You stumbled upon this project at the beginning of its development, so there's not much to see here.
This is for personal use, but I plan on making it configurable with yaml (or some other language) if you want to port it for your own use.
I am very much a beginner with C#, so development will be slow. If you notice anything in this code base that is sickening or just generally want to help develop this project, feel free to create an issue or pull request.
